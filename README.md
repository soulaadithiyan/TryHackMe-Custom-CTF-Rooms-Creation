# 🔐 Custom TryHackMe Cybersecurity Labs

A collection of custom-built Capture The Flag (CTF) labs inspired by real-world penetration testing scenarios. These projects were designed to provide hands-on experience in cybersecurity, ethical hacking, web exploitation, steganography, privilege escalation, and attack-chain simulation.

# 📌 Projects Included

## 1️⃣ Mini Web CTF – Web Exploitation Lab

A vulnerable web-based CTF environment focused on web application security and penetration testing techniques.

### 🔥 Key Features
- Authentication bypass
- File upload vulnerability
- Remote Code Execution (RCE)
- Reverse shell access
- Privilege escalation
- Flag-based challenge workflow

### 🛠 Technologies & Tools
- Ubuntu
- Apache2
- PHP
- MySQL
- Netcat
- VirtualBox

### ⚔ Attack Flow
1. Port Scanning
2. Directory Enumeration
3. Authentication Bypass
4. Malicious File Upload
5. Reverse Shell Access
6. User Enumeration
7. Privilege Escalation
8. Root Flag Retrieval

### 📚 Learning Outcomes
- Understanding insecure file upload vulnerabilities
- Practical web exploitation workflow
- Reverse shell handling
- Linux privilege escalation basics

---

## 2️⃣ StegoCTF – Steganography & Privilege Escalation Lab

A multi-stage CTF challenge involving steganography, password cracking, SSH authentication, and privilege escalation.

### 🔥 Key Features
- Anonymous FTP access
- ZIP password cracking
- Steganography-based hidden credential extraction
- SSH key authentication
- Linux privilege escalation

### 🛠 Technologies & Tools
- Ubuntu
- OpenSSH
- vsftpd
- Steghide
- JohnTheRipper
- fcrackzip

### ⚔ Attack Flow
1. FTP Enumeration
2. Download ZIP Archive
3. Crack ZIP Password
4. Extract Hidden SSH Key
5. SSH Login
6. Privilege Escalation
7. Root Flag Retrieval

### 📚 Learning Outcomes
- Practical steganography techniques
- Password cracking workflows
- SSH exploitation concepts
- Linux privilege escalation techniques
